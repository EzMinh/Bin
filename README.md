# How to use
- Use /bin help
# Permission
| Permission |        Description        | Default |
|:----------:|:-------------------------:|:-------:|
|   bin.cmd  | Allows player to use /bin |   True  |

Step 1. Make sure you installed Sheep plugin <br/>
Step 2. Run ```install Bin``` in console <br/>
Step 3. Enjoy this plugin
